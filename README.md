# FUTURE_FS_01 — Personal Portfolio Website

A modern, responsive personal portfolio website to showcase my skills, projects, and profile for internship evaluation and recruiter visibility.

## 🔗 Live Demo
- Live Site: ((https://future-fs-01-sd.vercel.app/))
- GitHub Repo: (this repository)

## 📌 Project Objective
Build and deploy a portfolio website that includes:
- Professional introduction
- Skills & tech stack
- Projects showcase
- Contact section (optional email form)
- Clean UI + responsive design

## ✨ Features
- ✅ Responsive design (mobile + desktop)
- ✅ About / Skills / Projects / Contact sections
- ✅ Modern UI using Tailwind CSS
- ✅ Easy to update content (components based)
- ✅ Fast build using Vite (React)

## 🛠 Tech Stack
- **Frontend:** React (Vite), JavaScript, HTML5, CSS3
- **Styling:** Tailwind CSS
- **Version Control:** Git & GitHub
- **Deployment:** Netlify / Vercel
